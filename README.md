# Hardware Store & Building Materials Catalog

A full-stack web application for a hardware store and building materials supplier.  
The project includes a **public product catalog** and a **secure admin panel** to manage categories, products, and images.

Designed for real businesses that want to showcase their products online **without displaying prices or stock**, redirecting customers to WhatsApp for direct inquiries.
